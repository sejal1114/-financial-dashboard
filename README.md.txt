# 📊 Financial Analytics Dashboard

An interactive full-stack dashboard for financial analysts to track, filter, visualize, and export company transactions with ease.

---

## 🚀 Features

- 🔐 JWT-based secure login
- 📈 Interactive charts for revenue, expenses, and category breakdowns
- 🧾 Transactions table with search, sort, and filter
- ⚠️ Alert chip-based error feedback
- 📤 Configurable CSV export (select columns, auto-download)
- 📱 Fully responsive, user-friendly interface

---

## 🎯 Use Case

Financial analysts need to:

1. Log in securely
2. View dashboards with visual trends and metrics
3. Search, filter, and sort transaction records
4. Export filtered data as configurable CSV reports

---

## 🛠️ Tech Stack

### 🔹 Frontend

- **React.js** with **TypeScript**
- **Recharts** for charting
- **Material UI / CSS modules** for design
- **React Router** for navigation

### 🔸 Backend

- **Node.js** with **TypeScript**
- **Express.js**
- **MongoDB** (Mongoose ODM)
- **JWT Authentication**
- **csv-writer / fast-csv** for CSV generation
